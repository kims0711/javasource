입출력(Input/Output)

- 기본입력: 키보드
- 기본출력: 화면

-System.out : 화면
-System.in : 키보드  
    > new Scanner(System.in);

- 입력장치(키보드, 파일, 객체...)에서 읽어오기 
    > text 파일 : 문자스트림
    > text + 모든 파일 : 바이트스트림

- 스트림(stream)
: 서로 연결된 두 대상의 연결통로
    > 스트림은 양방향 사용이 불가하다
    > ∴ 입력, 출력을 따로 처리함

- File : 자바에서 파일(텍스트,이미지,동영상...)처리를 담당하는 객체
    > 경로 구분자
        >> 윈도우: ; (세미콜론)
        >> 유닉스(리눅스): : (콜론)
    > 이름 구분자
        >> 윈도우: \  (폴더 구분자로 \ 쓰려면 두 개 써야함 -> \\)
        >> 유닉스(리눅스): /  (얜 하나만 써도 ㄱㅊ)

- 주요 메소드
    > 파일 정보 추출
        >> getName() : 파일명
        >> getAbsolutePath() : 경로포함 파일명
        >> getParent() : 부모 폴더
        >> File.separator(File.separatorChar) : 운영체제에 맞춰 구분자 생성
        >> length() : 파일 크기 추출
        >> lastModified() : 최종 수정일자

    > 파일 / 디렉토리 생성
        >> mkdir() : 만들고자 하는 디렉토리의 상위 디렉토리가 존재하지 않을 경우 생성 불가
        >> mkdirs() : 만들고자 하는 디렉토리의 상위 디렉토리가 존재하지 않을 경우 상위 디렉토리까지 생성  
        >> createNuewFile() : txt 파일 생성

    > 디렉토리 정보 추출
        >> isDirectory() : true 일 때
